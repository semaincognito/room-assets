# UI — Room Assets

## Навигация
В верхней части страницы находится Header с вкладками:
- Rooms
- Assets
- Bookings
- Data

## Rooms
Страница содержит:
- кнопку "Добавить аудиторию"
- таблицу аудиторий
- модальное окно создания аудитории (номер, название, вместимость)
- кнопку удаления аудитории в строке таблицы

Есть переключатель режима Rooms:
- CRUD (Task 1) — работа с localStorage
- Demo (MSW) — демонстрация загрузки данных через мок API

## Assets
Страница содержит:
- кнопку "Добавить ресурс"
- таблицу ресурсов
- модальное окно создания ресурса (название, описание)
- кнопку удаления ресурса

## Bookings
Страница содержит:
- кнопку "Создать бронь" (отключена, если нет Rooms)
- таблицу бронирований
- модальное окно создания брони:
  - выбор аудитории
  - название
  - start/end (datetime-local)
- при конфликте времени показывается ошибка

## Data
Страница содержит:
- кнопку Export JSON — скачивает файл с текущими данными
- кнопку Import JSON — загрузка JSON файла и применение данных
- сообщения об успехе / ошибке импорта
