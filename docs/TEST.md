# TEST

## Запуск
1) `npm install`
2) `npm run dev`
3) Открыть `http://localhost:5173`

## Rooms
1) Создать аудиторию → появилась в таблице
2) Создать аудиторию с таким же номером → ошибка/запрет
3) Поиск по номеру/названию → список фильтруется
4) Фильтр по минимальной вместимости → список фильтруется
5) Удаление аудитории → исчезла
6) Перезагрузка страницы → данные сохранились

## Assets
1) Создать ресурс → появился в таблице
2) Создать ресурс с таким же названием → ошибка/запрет
3) Поиск по названию → фильтрация
4) Удаление → исчез
5) Перезагрузка → данные сохранились

## Bookings
1) Создать бронь (валидные start/end) → появилась
2) Создать бронь с end <= start → ошибка
3) Создать бронь с пересечением времени в одной аудитории → ошибка конфликта
4) Фильтр по аудитории → список фильтруется
5) Удаление → исчезла
6) Перезагрузка → данные сохранились

## Data import/export
1) Export JSON → файл скачан
2) Очистка данных (удалить записи) → таблицы пустые
3) Import `seed.example.json` → данные восстановились
4) Импорт неверного JSON → сообщение об ошибке

## MSW Demo режим
1) Rooms → переключить режим Demo (MSW)
2) Данные загружаются через `GET /api/rooms`
3) Принудительно вернуть `status: 500` в handler → показывается ошибка
